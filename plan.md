# Final Project – Midterm Plan

## Deploy Link
https://wp-final-tau.vercel.app/

## 1. 本次 Prototype 已完成

### 核心功能
- **用戶認證系統**
  - 用戶註冊與登入
  - 使用 Supabase Auth 進行身份驗證
  - Session 管理與中間件處理

- **食譜上傳功能**
  - 完整的食譜表單（標題、描述、封面圖片）
  - 動態食材列表（支援分類、單位、備註）
  - 步驟管理（支援每個步驟上傳圖片、設定計時器）
  - 圖片壓縮與優化（使用 Cloudinary）
  - 進度指示器與詳細的發布流程 UX
  - 食譜編輯功能

- **食譜瀏覽與探索**
  - 食譜列表展示
  - 食譜詳情頁面
  - 食譜卡片（顯示評分、收藏數、時間、難度等）
  - 搜尋功能

- **社交互動功能**
  - 食譜評分系統（1-5 星）
  - 收藏/取消收藏食譜
  - 留言系統（查看、新增、編輯、刪除）
  - 個人資料頁面（顯示用戶資訊、上傳的食譜、收藏的食譜）
  - Tab 切換（食譜/收藏）

- ✅ **烹飪模式**
  - 步驟式引導介面
  - 計時器功能


## 2. 最終版本預計完成項目

### 核心功能增強
- **用戶追蹤系統**
  - 關注/取消關注其他用戶
  - 追蹤者列表與正在關注列表
  - 動態消息流（顯示關注用戶的新食譜）

- **進階搜尋與過濾**
  - 多條件搜尋（食材、標籤、難度、時間等）
  - 搜尋結果排序（最新、最熱、評分最高）
  - 標籤系統優化與分類管理
  - 收藏夾管理（自訂收藏夾名稱）

- **食譜改進**
  - 食譜版本管理（記錄修改歷史）
  - 草稿自動儲存功能
  - 食譜分享連結（社群媒體分享）
  - 列印友好版本
  - 食譜匯出功能（PDF、JSON）

### 新功能模組
- **冰箱剩食推薦**
  - 用戶輸入現有食材
  - AI 推薦可製作的食譜
  - 食材替代建議

- 🆕 **影音擴充**
  - 步驟影片上傳
  - 食譜製作影片
  - 影片播放器整合

- 🆕 **進階 AI 推薦**
  - 個人化推薦算法
  - 基於用戶行為的推薦
  - 相似食譜推薦
  - 食材搭配建議

### 技術優化
- 🔄 效能優化（圖片懶加載、分頁、快取）
- 🔄 SEO 優化（Meta tags、Structured Data）
- 🔄 錯誤監控與日誌系統
- 🔄 單元測試與整合測試
- 🔄 API 速率限制
- 🔄 無障礙功能 (a11y) 改善
